11 INICIAL REFERENCIAS

¿CÓMO SE HA HECHO?

Tecnologías utilizadas

Se han utilizado HTML5 para crear la estructura y el diseño de la página, poniendo etiquetas semánticas como header, main, section y footer.

Métodos de maquetación

Se ha aplicado CSS para organizar el contenido y hacer que el diseño sea adaptable a diferentes tamaños de pantalla.

Herramientas de desarrollo

Se han usado un editor de código, un navegador web y las DevTools para comprobar y ajustar el resultado.

Proceso de trabajo

El proceso ha incluido la planificación del diseño, la creación del código HTML y CSS, y la comprobación final para asegurar el correcto funcionamiento.

Decisiones de diseño

Se han elegido colores verdes y tonos claros, una tipografía sencilla y una estructura ordenada con cabecera, contenido y pie de página.
